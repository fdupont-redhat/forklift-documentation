Installing Forklift using the web console

You can install Forklift using the OpenShift web console.

Prerequisites

  • Install OpenShift Kubernetes Engine 4.6 on your cluster.

  • Install the KubeVirt Operator.

  • Log in as a user with cluster-admin privileges.

Procedure

  1. Open a browser window and log in to the OpenShift web console.

  2. Click Operators → OperatorHub.

  3. Use the Filter by keyword field to find the Konveyor Forklift Operator.

    Note

    The Konveyor Forklift Operator is a Community Operator. Red Hat does not support Community Operators.

  4. Click the Konveyor Forklift Operator and then click Install.

  5. On the Install Operator page, click Install.

  6. Click Operators → Installed Operators to verify that the Konveyor Forklift Operator appears in the openshift-migration project with the status Succeeded.

  7. Click the Konveyor Forklift Operator.

  8. Under Provided APIs, locate the ForkliftController, and click Create Instance.

  9. Click Create.

  10. Click Workloads → Pods to verify that the Forklift pods are running.

forklift-documentation is maintained by konveyor.
